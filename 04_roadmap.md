## The 4-step loop (what data science actually is)

Every project, every session, is the same loop:

1. **Ask** a clear question.
2. **Get** data that could answer it.
3. **Look** at the data — clean, chart, notice what's strange.
4. **Decide** what it tells you, and what it doesn't.

Tools (Python, pandas, LLMs) are helpers for these four steps. The steps are the craft.

## The modern way

- AI as **copilot**, not autopilot.
- Real project from day one.
- Concepts learned *when needed*, not before.
- Small wins, repeated, until confidence is real.

## 6-month shape

- **Month 1** — Foundations. Cloud setup. First data loaded. Chapter 3 written.
- **Month 2** — Text cleaning, word counts, first real chart on chosen sub-question.
- **Month 3** — Embeddings. Semantic comparison across two traditions.
- **Month 4** — LLM-assisted analysis. First fine-tuning experiment.
- **Month 5** — Writing up findings. Polishing notebooks. Public repo.
- **Month 6** — Final report + fine-tuned model on Hugging Face.

## What success looks like

Not a finished field of study. A finished *project*. One thing I made, start to end, that is mine. That I can explain to anyone. That lives on GitHub forever.
